🚀 COCBot - Istruzioni

1. ✅ Requisiti:
- Windows
- Python 3.10 o superiore
- Tesseract OCR

2. 📥 Installazione Tesseract OCR:
- Scarica da: https://github.com/tesseract-ocr/tesseract/releases
- Durante l'installazione, SPUNTA "Add to PATH"
- Se non lo fai, modifica la riga nel file script.py con il percorso corretto

3. 📂 Estrai la cartella ZIP da qualche parte

4. 🔧 Apri il terminale nella cartella estratta

5. ⬇️ Installa i pacchetti:
    pip install -r requirements.txt

6. ▶️ Avvia il bot:
    python script.py

7. ⛔ Per interrompere lo script:
    premi ESC

Il bot farà tutto in automatico, ti avviserà su Telegram quando trova un buon numero!
